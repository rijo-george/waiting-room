Features

Everything falls through the cracks.
Until now.

⇄

Two-Panel View

Left panel: what you're waiting on. Right panel: what others need from you. Always balanced, always visible.

🔴

Age Indicators

Green, yellow, red. Items age visually so the oldest open loops scream for attention before you forget them.

📬

Nudge Tracking

Log every follow-up. "Sent reminder email." "Pinged on Slack." Build a paper trail without leaving the app.

✅

Resolve & Archive

Done? Resolve it. Every item moves to history with timestamps and duration. See your track record over time.

🎨

6 Themes

Dark, Light, Sunset, Ocean, Forest, Rose. Pick your vibe. Themes sync between the macOS app and the terminal.

🔒

100% Local

All data lives in ~/.waiting-room/. No accounts. No cloud. No tracking. Just a JSON file you own.

How It Works

Stupid simple.
By design.

1

Add an open loop

Press A. Type who and what. Optional: expected date and a note. That's it.

2

Watch it age

Green (fresh), yellow (getting old), red (overdue). The color tells you what needs attention before you even read it.

3

Nudge or resolve

Followed up? Press N to log it. Done? Press R to resolve. It moves to history with a timestamp.

Get started in
30 seconds

No accounts. No install wizards. Just clone and run.

Terminal 
# Clone the repo
git clone https://github.com/rijo-george/waiting-room.git
cd waiting-room/macos

# Build and run
bash build-app.sh
open "The Waiting Room.app"
Terminal 
# Clone the repo
git clone https://github.com/rijo-george/waiting-room.git
cd waiting-room/tui

# Install dependency
pip install textual

# Run
python3 waiting_room.py
All data stored locally at ~/.waiting-room/data.json. Both apps share the same file.
